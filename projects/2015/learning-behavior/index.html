Learning behaviors to optimize the player’s experience in Robogames

 PhD Research

 Follow code

The project investigates how to develop probability-based models for enabling autonomous robots to better understanding and select actions in a Physically Interactive RoboGame. Specifically, it focuses on the development of a player model (based on Latent Dirichlet Allocation) that represents the robot's adversaries (human players) as a probability mixture proportion of uncovered types (topics) from data. With the model at hand, we desire to establish a self-adjusting autonomous agent capable of selecting actions towards matching the player’s current gameplay (type) for the purpose of sustaining/increasing interaction and, therefore, foster entertainment.
